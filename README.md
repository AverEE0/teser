# Веб-приложение для заполнения договора

Это веб-приложение позволяет автоматически заполнять шаблон договора на основе данных из фото или документа.

## Функциональность

- Загрузка шаблона договора (DOCX с плейсхолдерами)
- Загрузка источника данных (фото для OCR или DOCX для парсинга)
- Извлечение ключевых данных: ФИО, наименование имущества, адрес, стоимость услуг, подпись
- Ручная корректировка извлеченных данных
- Генерация и скачивание заполненного договора в формате DOCX

## Технологии

- HTML, CSS, JavaScript
- Tesseract.js для OCR
- Mammoth.js для парсинга DOCX
- docxtemplater для заполнения шаблонов
- FileSaver.js для скачивания файлов

## Запуск

Откройте `index.html` в браузере (Chrome, Firefox, Edge).

## Шаблон

Шаблон должен быть DOCX файлом, где поля для замены выделены желтым цветом (highlight yellow). Порядок выделений должен соответствовать: ФИО, наименование имущества, адрес, стоимость услуг, подпись заказчика.
